## Get Started

Make sure you have NodeJS >10.4 installed.

To make sure all Node modules are installed, run:

```
npm install
```

To run the tests with [Jest][jest], run:

```
npm run test
```

To generate documentation with [TypeDoc][typedoc], run:

```
npm run doc
```
