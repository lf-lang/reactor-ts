      The reactor containing this action.

      Optional. If physical, then the hardware clock on the local platform is used to determine the tag of the resulting event. If logical, the current logical time (plus one microstep) is used as the offset.

      Optional. Defaults to 0. Specifies the intrisic delay of any events resulting from scheduling this action.

get

  • get(): undefined | null

  • Inherited from Action.get

    • Defined in reactor.ts:473

    Called on an action within a reaction to acquire the action's value. The value for an action is set by a scheduled action event, and is only present for reactions executing at that logical time. When logical time advances, that previously available value is now unavailable. If the action was scheduled with no value, this function returns null.

    Returns undefined | null
